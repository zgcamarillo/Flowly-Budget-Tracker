<h1>Welcome to Flowly</h1>
<p>Flowly is an interactive budgeting application that allows users to track their
income, expenses, and available funds. It uses object oriented Javascript to manage the logic,
SCSS for animated gradients, custom styling, and a polished UI.</p>

<h2>Features</h2>
<ul>
  <li>Budget Managment</li>
  <li>Welcome Popup - Application displays a personal greeting, while also using a build in date formatting for a full day and weekday.</li>
  <li>Income Tracking - User is able to add income entries with title, amount, optional description and pay frequency</li>
  <li>Automatic Update - With every entry, total income, total expense, logs and available balance updates automatically.</li>
</ul>

<h2>Technology Used</h2>
<ul>
  <li>HTML</li>
  <li>SASS/CSS</li>
  <li>JS(ES6+)</li>
  <li>CSS Media Queries</li>
</ul>

<h2>How to Use</h2>
<ul>
  <li>Welcome - When entering the application, the user will be greeted with a welcome popup. As the used enters their name, after clicking submit they will be prompted to a personalized and easy to use budget tracker</li>
  <li>Add Your Income - In order to add income, the user will press the "+ ADD INCOME " button and the application will prompt the user to add a title, amount and an optional description. After
  filling out required areas, user will press "+ ADD" and your income will show up in the log for the users records.</li>
  <li>Add Your Expenses- In order for the user to add expenses, the user is to press "+ ADD EXPENSES" button and the application will then prompt the user to add a title, amount
  and optional description. After filling out required areas, user will oress "+ ADD" and expenses will log for the users records.</li>
  <li>Automatic Updates - Will automatic updates, any of the users entries will be calculated and displayed immediately, and Available Balance and Amount Spent will be displayed, allowing the user to see 
  how much money they will have available and how much money has been spent.</li>
</ul>
